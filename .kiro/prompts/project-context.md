# PathoMap Project Context

This is a Next.js project called PathoMap — an interactive mind map / visual tool related to the Pathologic game franchise by Ice-Pick Lodge.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Key Files
- `src/app/page.tsx` — Main page
- `src/app/MindMap.tsx` — Core mind map component
- `src/app/icons.ts` — Icon definitions
- `src/app/layout.tsx` — Root layout
- `public/icons-black/`, `public/icons-white/` — Game icon assets

## Important
Read `AGENTS.md` at project root before writing any Next.js code — this version may have breaking changes from your training data. Check `node_modules/next/dist/docs/` for current API docs.
